README - Wood Qi Pill Pack (ACSM_MaxQiPill)
=============================================

สิ่งที่ม็อดนี้ทำ
----------------
เพิ่มไอเทมยา 4 ระดับ "木灵丹" (Wood Spirit Pill) ที่คราฟจากเตาเล่นแร่แปรธาตุ (Alchemy Furnace)
โดยใช้ไม้เป็นวัตถุดิบ กินแล้วเพิ่ม Max Qi แบบถาวร:
  - Tier 1: +1,000 Max Qi
  - Tier 2: +10,000 Max Qi
  - Tier 3: +100,000 Max Qi
  - Tier 4: +1,000,000 Max Qi

ติดตั้งยังไง (มือถือ)
----------------------
1. คัดลอกโฟลเดอร์ ACSM_MaxQiPill ทั้งโฟลเดอร์นี้ ไปไว้ที่:
   /storage/emulated/0/Android/data/com.GSQ.XiaWorld.googleplay/files/Mods/
   (ถ้าไม่มีโฟลเดอร์ Mods ให้สร้างเอง)
2. เปิดเกม เข้าเมนู Mod Manager แล้วเปิดใช้งานม็อดนี้
3. เริ่ม/โหลดเซฟใหม่ แล้วตรวจดูใน Log ว่าม็อดโหลดสำเร็จหรือมี error

จุดที่ "มั่นใจว่าถูก" (ตรงตามเอกสาร modding ที่มีอยู่จริง)
---------------------------------------------------------
- Settings/Modifiers/...xml  → ตัวบัฟ Max Qi ทั้ง 4 ระดับ
- Settings/Esoterica/...xml  → ตำราที่ปลดล็อกสูตรยา
- Settings/Language/...xml   → คำแปลอังกฤษของไอเทม
- โครงสร้างไฟล์/โฟลเดอร์ Mods ทั่วไป, Info.json

จุดที่ "ยังไม่ยืนยัน 100%" (ต้องทดสอบ/ปรับเอง)
-------------------------------------------------
1. Settings/ThingDef/Item/...xml → ส่วน <Drug><Modifiers>...</Modifiers></Drug>
   (ตรงนี้คือ "กินแล้วเกิดอะไรขึ้น" - ผมหาเอกสารยืนยัน tag ที่แน่ชัดไม่ได้ แม้แต่คู่มือ
   modding ที่ละเอียดที่สุดก็ยังเขียนว่าส่วนนี้ "ยังไม่มีคนสรุป" ไว้)
2. Settings/Practice/DanFormula/...xml → สูตรคราฟ (ใช้ไม้เท่าไหร่ ต้องใช้เตาตัวไหน)
   ส่วนนี้เป็น "ของตัวอย่างคร่าวๆ" ไม่ใช่ของจริง ต้องแก้

วิธีแก้ 2 จุดนี้แบบชัวร์ที่สุด (แนะนำ)
------------------------------------------
ก. เปิดไฟล์เกมจริงด้วย File Manager (ต้องขอสิทธิ์เข้าถึงไฟล์ทั้งหมด):
   /storage/emulated/0/Android/data/com.GSQ.XiaWorld.googleplay/files/Settings/
   หาไอเทมยาเพิ่ม Max Qi ที่มีอยู่แล้วในเกม เช่น
   Item_Dan_SanHuaHuangJing (Trimerous Essence Pill)
   ก็อปปี้ทั้งบล็อก <ThingDef>...</ThingDef> ของมันมาดูว่า <Item><Drug>... จริงๆเขียนยังไง
   แล้วก็อปสูตรคราฟของมันจาก Settings/Practice/DanFormula/Dan.xml มาดูโครงสร้างจริง
   จากนั้นแก้เฉพาะชื่อไอเทม คำอธิบาย Modifier ที่อ้างถึง และวัตถุดิบ (เปลี่ยนเป็นไม้)
   วิธีนี้การันตีว่าไม่ Error เพราะอ้างจากของที่ใช้งานได้จริงในเกม

ข. หรือง่ายกว่านั้น: ใช้ตัวสร้างไอเทมในเกมเอง (Content Editor / Realms Palace)
   ที่มี UI ให้เลือกผลของยา (เพิ่ม Max Qi เท่าไหร่), เลือกอาคารที่ใช้คราฟ (เตาเล่นแร่แปรธาตุ),
   และเลือกวัตถุดิบ (ไม้) ได้เลยโดยไม่ต้องเขียน XML เดามือเองสักตัว - เหมาะกับการทำของแบบนี้
   มากเพราะตรงตามที่ขอเป๊ะ และไม่มีความเสี่ยงเรื่อง tag ผิด

ทดสอบเร็วๆแบบไม่ต้องเล่นสูตรคราฟ
-----------------------------------
ถ้าอยากเทสว่าไอเทม+Modifier ทำงานถูกไหมก่อน โดยไม่ต้องพึ่ง DanFormula เลย
ใช้ม็อด MODModifier (ตัวโกง/แอดมิน) สั่ง spawn ไอเทม Item_Dan_MuLingDan_T1 ให้ตัวละครกินดูได้เลย
ถ้า Max Qi ขึ้น 1,000 แปลว่าไอเทม+Modifier ใช้งานได้แล้ว เหลือแค่ทำสูตรคราฟให้ถูกอีกจุดเดียว
